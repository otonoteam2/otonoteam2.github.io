How the random paper works (Maths Bridge SAM set 1):
• The system selects 4 random sections. Each section is worth 20 marks, so the full paper is always 80 marks.
• Section A — GCSE fluency: basic algebra, substitution, expanding, factorising, indices. 20 marks.
• Section B — GCSE+ bridge: straight lines, functions, inverses, composites, rearranging. 20 marks.
• Section C — First A-level skills: basic differentiation and gradient functions. 20 marks.
• Section D — A-level bridge challenge: basic integration, definite integration and mixed harder questions. 20 marks.
• The paper is ordered easy at the front and hard at the back, just like a proper bridge assessment.
